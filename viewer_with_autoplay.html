  1. GitHub 로그인 → 우측 상단 프로필 → Settings
  2. 맨 아래 Developer settings → Personal access tokens → Fine-grained tokens → Generate new token
  3. Repository access에서 -hs_thinking_basic_library_viewer 저장소만 선택
  4. Repository permissions → Contents를 Read and write로 설정 후 생성
  5. 생성된 토큰(github_pat_…)을 위 칸에 붙여넣고 저장하세요.

토큰이 부담되면: "코드만 복사" 버튼을 눌러 나온 코드를 복사한 뒤, 도와주는 사람(또는 Claude)에게 전달하면 대신 반영해 드릴 수 있습니다.
